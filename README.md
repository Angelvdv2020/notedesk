# NoteDesk Office — PWA

## Файлы

Все 5 файлов должны лежать в одной папке:

```
notedesk.html   — приложение
manifest.json   — PWA-манифест
sw.js           — Service Worker (офлайн-кэш)
icon-192.png    — иконка 192×192
icon-512.png    — иконка 512×512
```

## Деплой на GitHub Pages (бесплатно)

1. Создайте репозиторий на github.com (например `notedesk`)
2. Загрузите все 5 файлов в корень репозитория
3. Settings → Pages → Source: `main` branch, root `/`
4. Через минуту сайт будет доступен по адресу: `https://ваш_логин.github.io/notedesk/notedesk.html`

## Деплой на Netlify (бесплатно)

1. Перетащите папку с 5 файлами на netlify.com/drop
2. Готово — получите ссылку вида `https://random-name.netlify.app/notedesk.html`

## Установка как приложение

- **Android Chrome**: откройте ссылку → меню ⋮ → "Установить приложение"
- **iOS Safari**: откройте ссылку → кнопка "Поделиться" → "На экран Домой"
- **Desktop Chrome/Edge**: иконка ⬇ в адресной строке или в статус-баре NoteDesk

## Что даёт PWA

- Работает офлайн (все данные в IndexedDB на устройстве)
- Иконка на домашнем экране
- Без адресной строки — выглядит как нативное приложение
- Автообновление при выходе новой версии
- Кнопка "Установить" в статус-баре NoteDesk
